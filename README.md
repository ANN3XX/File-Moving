# File-Moving
Automatically move files from one folder to another, with a certain extension (in this case '.html').

# Usage
<ol>
  <li>Copy the code from 'main.py' to your own Python file</li>
  <li>Make sure to replace "/path/to/source/folder" and "/path/to/destination/folder" with the actual paths of your source and destination folders.</li>
</ol>
